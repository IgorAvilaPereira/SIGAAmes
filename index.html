Para usar, entre no SIA, acesse Período Letivo >> Início do Período >> Planejamento >> Planejamento das Disciplinas, selecione a disciplina e clique em alterar (ícone do lápis) e cole seu planejamento na caixa abaixo usando campos separados por ponto-e-vírgula conforme o exemplo a seguir:

1;10/03/2017;Atividades para este horário;
1;10/03/2017;Atividades para este horário;
1;17/03/2017;Atividades para este horário;

Os dados são, respectivamente, o período (ciclo ou bimestre), data (no formato dd/mm/aaaa) e a descrição da atividade.

Para concluir, clique em "Importar" e veja a mágica acontecer ;)

Caso deseje remover todas as atividades planejadas, clique em "Limpar".
